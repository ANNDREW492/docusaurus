## 🎨 Editar estilos en tiempo real
1. Selecciona un elemento en el inspector.
2. En la pestaña **Styles**:
   - Marca/desmarca reglas CSS.
   - Añade nuevas propiedades con `+`.

### 📊 Visualizar modelo de caja
- Usa la pestaña **Computed** para ver:
  - Margen, padding, bordes.
  - Dimensiones exactas del elemento.
